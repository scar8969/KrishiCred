## Revenue Streams

### 1. Transaction Fee on Stubble Routing

| Parameter | Value |
|-----------|-------|
| Fee | ₹50/ton |
| Typical transaction | 4 tons/acre |
| Revenue per acre | ₹200 |
| Margin | 18% (after UPI commissions, operations) |

### 2. Carbon Credit Commission

| Parameter | Value |
|-----------|-------|
| Commission rate | 15% of credit value |
| Credit value | ₹400/ton CO2e |
| Typical credits per acre | ~2 tons CO2e |
| Revenue per acre | ₹120 |
| Margin | 85% |

### 3. SaaS Subscription for Plants

| Tier | Price | Features |
|------|-------|----------|
| Basic | ₹50,000/year | Matching, alerts, basic dashboard |
| Premium | ₹1,50,000/year | Priority routing, advance booking, analytics |
| Enterprise | Custom | Dedicated account, API access |

### 4. Government Data Licensing

| Product | Price | Customers |

| Fire hotspot data | ₹25L/year | State pollution boards |
| Compliance reports | ₹10L/year | District administrations |
| Impact verification | ₹15L/year | Central ministries |

### 5. ESG Premium Certification

| Service | Price | Value to Customer |
|----------|-------|-------------------|
| "Carbon-Negative Biogas" label | ₹5L/plant | Premium pricing for CBG |

  • Satellite & API             ₹23
  • Field operations            ₹25

Contribution Margin:            ₹150

Break-even Volume: 16,667 acres/month
