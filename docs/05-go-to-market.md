
## Sales Strategy

### Sales Process

```
LEAD GENERATION
├── Farmer: Field agents, WhatsApp, referrals
├── Plant: Direct outreach, industry events
└── Carbon Buyer: Marketplace, direct outreach

QUALIFICATION
├── Farmer: Land ownership verification
├── Plant: Capacity verification, location
└── Carbon Buyer: ESG requirements, budget

PROPOSAL
├── Farmer: WhatsApp quote with pickup date
├── Plant: Supply agreement with pricing
└── Carbon Buyer: Credit portfolio with verification

CONVERSION
├── Farmer: WhatsApp confirmation
├── Plant: Contract signing
└── Carbon Buyer: Marketplace transaction

ONBOARDING
├── Farmer: Land record verification, UPI setup
├── Plant: API integration, dashboard setup
└── Carbon Buyer: Wallet setup, KYC
```
