KrishiCred API 1.0.0 OAS 3.1 AI-powered climate finance platform for crop residue management. Key Endpoints: - /api/v1/firewatch: Alerts, stats, satellite processing. - /api/v1/stubble: Routes, optimization, nearby farms, plant capacity. - /api/v1/carbon: Credit calculation, creation, verification, certification, sales. - /api/v1/farmers: Registration, location updates, alerts, stats. - /api/v1/plants: Management, stock updates, routes, stats. Schemas: AlertResponse, BiogasPlantResponse, CarbonCreditResponse, FarmerResponse, FireAlertResponse, RouteResponse, etc.
